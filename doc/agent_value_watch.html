init/2

init(Agent::agent:agent_ref(), Fun::agent:watch_fun()) -> none()

Initialises the watcher process. Before jumping to the main loop it starts monitoring the agent process.

loop/2

loop(Agent::agent:agent_ref(), Fun::agent:watch_fun()) -> none()

The watcher process main message loop. It's capable of receiving two messages: {'DOWN', _, process, AgentPid, _} and {'agent-value', agent:agent_ref(), OldValue, NewValue}. If the evaluation raises an exception of any kind the process exits sending an error report to the error_logger.
